DASHBOARD BUDDY

Comment lancer le projet :

1. Lancer le script create_db.sql qui se trouve dans le dossier database en utilisant un logiciel comme MySQL Workbench, par exemple.
2. Utiliser un logiciel comme XAMPP Control Panel pour établir une connexion MySQL.
3. Ouvrir le projet et exécuter npm install dans le terminal.
4. Exécuter la commande: .\tunnelto-windows.exe set-auth --key 17YIxABVdVrtjjAezDXGPO
5. Exécuter la commande: .\tunnelto-windows.exe --subdomain dashboard-buddy --port 3000
6. Se déplacer dans le dossier backend et exécuter npm run watch
7. Ouvrir l'application web sur http://localhost:3000/ ou sur https://dashboard-buddy.tunnelto.dev.
