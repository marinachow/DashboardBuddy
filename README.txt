DASHBOARD BUDDY

Comment lancer le projet :

1. Lancer le script create_db.sql qui se trouve dans le dossier database en utilisant un logiciel comme MySQL Workbench, par exemple.
2. Utiliser un logiciel comme XAMPP Control Panel pour créer une connexion MySQL.
3. Ouvrir le projet et exécuter npm install dans le terminal.
4. Se déplacer dans le dossier backend et exécuter node .\server.js.
5. Ouvrir l'application web sur http://localhost:3000/.
